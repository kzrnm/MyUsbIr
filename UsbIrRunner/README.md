## 概要

[USB赤外線リモコンアドバンス](http://bit-trade-one.co.jp/product/module/adir01p/)に送信する実行ファイルです。


## 使用方法

- `UsbIrRunner.exe <--file> filepath [<-f|--freq> frequency]`
- `UsbIrRunner.exe <-b|--base64> base64String [<-f|--freq> frequency]`

上記のいずれかの方法で実行します。

ファイルを指定した場合は読み込んだバイナリをそのまま送信データとします。

## 環境

.NET Framework 3.5以上を想定しています
